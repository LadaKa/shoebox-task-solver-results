# Shoebox Task Solver — Experiments & Results

This repository contains experiments and results for autonomous task solving in robot-assisted education (Shoebox Task Solver).

The work focuses on generating PDDL-based models from visual task representations, and testing their use in dynamic planning scenarios.

---

## 📁 Demonstrations (2026)
Conceptual proof-of-concept tests.

### Domain Model Generation
Ininitial experiments on generating PDDL domain models directly from images of tasks using ChatGPT-5.

- Path: `./demonstrations/2026/Domain_Model_Generation`

### Initial Dynamic Planning
Early experiments on integrating generated PDDL problems into planning and execution workflows.
Presented at RIE 2026 and AICCONF 2026.

- Path: `./demonstrations/2026/Initial_Dynamic_Planning`
